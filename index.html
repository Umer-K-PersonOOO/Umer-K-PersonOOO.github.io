Talented researchers struggle every day with using their computers.

Problem #1: The Shell

Most research software still runs on a shell program from the 1980s. User-friendliness was never a priority of bash, but researchers still have to use it daily.

Problem #2: The Code

Coding skills are a huge roadblock for scientists looking to transition into computational, data-driven research. Intelligent people have intelligent ideas that they struggle to convey to a machine.

Problem #3: The Packaging

Software developers struggle to package their code for cross-platform distribution, and their users struggle with installation and environment setup. Too much time is wasted just getting programs to run.

We founded Neural Computer Interfaces to solve this problem.
