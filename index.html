Axon Terminal

Welcome to the Axon Terminal!

Axon is a command line with a built in AI assistant.
Key features include:


Provides a description of each command you run

Chatbot that caters to both general research inquiries and specific support needs

Copilot that execute python code to perform various tasks on your system given plaintext instructions

Why use Axon?

Be more accurate. Axon informs users when executing potentially dangerous instructions. You always know what you are running, preventing risks to system integrity and data.
Built-in Search Engine. No need to search through Google and Stack Overflow when you can ask questions as you work.
Save time. No more wasting time reading through documentation, setting up environments, or installing packages. Simply tell Axon what you need to do!

How do I Install Axon?

Axon is currently in Alpha release as a ZSH plugin for oh-my-zsh. To request an API key, contact email@example.com. Visit the GitHub page for installation instructions.
